# Financial Health Analysis using SQL

## Project Overview
This project analyzes the financial health of countries using economic indicators such as GDP, inflation, interest rate, and domestic credit.

## Tools Used
- MySQL
- SQL (Views, Window Functions)
- Tableau

## Key Steps
- Data Cleaning & Unpivoting
- Normalization using Min-Max method
- Reversing negative indicators
- Weighted Financial Health Score
- Country Ranking and Trend Analysis

## Key Insights
- Identified strongest and weakest economies year-wise
- Tracked economic improvement and decline using trends
- Measured stability using volatility analysis

## Outcome
Prepared clean analytical datasets for dashboard visualization and decision-making.
